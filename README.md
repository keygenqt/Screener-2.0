Screener 2.0
===================

A terminal app for take screenshot

{version} - 0.0.1

* take desktop
* take area
* dealy
* png/jpg

### Usage

You can build *.pyz

```shell
git clone https://github.com/keygenqt/screener-2.0.git
cd screener-2.0
chmod +x ./build.sh
./build.sh
python3 screener-{version}.pyz
```

#### dependency:

```shell
pip3 install shiv
sudo apt-get install scrot
sudo apt-get install xclip
```
